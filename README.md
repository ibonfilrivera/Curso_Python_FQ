# Curso de Python - Facultad de Química (UNAM)

Repositorio oficial con el material didáctico, códigos y bases de datos empleados durante el curso de Python.

## 📅 Estructura del Curso
* **Sesión 1:** Introducción a la sintaxis de Python, tipos de datos y estructuras de control.
* **Sesión 2:** Uso de librerías numéricas (NumPy) y manejo de datos con Pandas.
* **Sesión 3:** Visualización de datos (Matplotlib/Seaborn) y aplicaciones químicas.

## 📊 Datos utilizados
* `curated_solubility.csv`: Dataset curado de solubilidad acuosa de compuestos orgánicos.
